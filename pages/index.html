Pie Print Server v1.6.1 - Basic API Usage

  • [GET] • /
    • Halaman ini.

  • [GET] • /printer-list
    • Mendapatkan list printer yang tersedia.

  • [GET] • /printer-default
    • Mendapatkan printer default.

  • [POST] • /print-url
    • Cetak menggunakan URL PDF.
    • Body : text/json 
    • {
    url: "http://..." [required]
    options: { [optional]
        (pdf-to-printer options)
    }
}

  • [POST] • /print-url-env
    • Cetak menggunakan URL PDF dengan konfigurasi .env file.
    • Body : text/json 
    • {
    url: "http://..." [required]
    options: { [optional]
        (pdf-to-printer options)
    }
}

  • [GET] • /env
    • Cek konfigurasi .env file.

  • [GET] • /test-print
    • Test print dengan printer default.

  • [GET] • /test-print-env
    • Test print dengan konfigurasi .env file.

  • refer to pdf-to-printer project repository for more information.
